# Under the Brambles

A place for mossy paths, drying herbs, well-worn recipes, and timeless crafts.

## Development

```
bundle exec jekyll server
```

## Logo

The logo is created with Piskel: https://www.piskelapp.com/

## Images

1. Download PNG images: https://www.cleanpng.com/
2. Resize image to fit a canva with the biggest side being equal to 500 px: https://squoosh.app/
3. Convert images to pixel art (pixel size 15) and download the small version: https://pixelartvillage.com/
